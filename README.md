# Plise Perde Sistemleri

Türkiye'nin en güvenilir plise perde üreticisi için oluşturulmuş Next.js tabanlı profesyonel web sitesi.

## Özellikler

- **Next.js 14** - En hızlı React framework
- **Tailwind CSS** - Modern ve responsive tasarım
- **SEO Optimized** - Schema markup ve structured data
- **Mobile First** - %100 mobil uyumlu
- **Performance** - Lighthouse %100 hedefli
- **Türkçe İçerik** - Tamamen Türkçe ve SEO uyumlu

## Kurulum

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Üretim için build et
npm run build

# Üretim sunucusunu başlat
npm start
```

## Proje Yapısı

```
pliseperdesistemleri/
├── pages/              # Next.js sayfaları
│   ├── index.js       # Ana sayfa
│   ├── urunler/       # Ürünler sayfaları
│   ├── hizmetler/     # Hizmetler sayfaları
│   ├── hizmet-bolgeler/ # Hizmet bölgeleri
│   ├── bayilik.js     # Bayilik sayfası
│   └── iletisim.js    # İletişim sayfası
├── components/        # React bileşenleri
│   ├── Header.js      # Üst navigasyon
│   ├── Footer.js      # Alt kısım
│   └── MobileFooter.js # Mobil sabit footer
├── styles/           # CSS dosyaları
│   └── globals.css   # Global stiller
├── public/           # Statik dosyalar
│   ├── sitemap.xml   # XML sitemap
│   ├── robots.txt    # Robots.txt
│   ├── llms.txt      # LLMs bilgisi
│   └── ai.txt        # AI bilgisi
└── package.json      # Bağımlılıklar
```
